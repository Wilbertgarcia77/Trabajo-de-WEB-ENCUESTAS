# Trabajo-de-WEB-ENCUESTAS
Esta es una aplicación web pensada para gimnasios y sus clientes, con la idea de que cada persona pueda tener un seguimiento más personalizado de su entrenamiento y nutrición. La app cuenta con un login, así que cada usuario tiene su propio acceso y el gimnasio puede llevar un registro de todos sus clientes de forma organizada.

Cuando un usuario inicia sesión, aparece su nombre en pantalla y la app lo guía a través de pequeñas encuestas. Estas preguntas son sobre datos personales y hábitos, y nos ayudan a saber si la persona está lista para empezar a entrenar y qué tipo de ejercicio sería más adecuado para ella. Según las respuestas, la inteligencia artificial de OpenAI genera recomendaciones personalizadas, como cuántas horas de ejercicio son adecuadas, qué tipo de entrenamiento le conviene más, o si necesita empezar con algo más básico antes de avanzar.

La app no se queda solo en los ejercicios: también incluye un apartado de nutrición, donde la inteligencia artificial puede dar sugerencias sobre alimentación, hábitos saludables y planes adaptados a cada persona. Todo esto se guarda en una base de datos, así que cada reporte, recomendación o progreso queda registrado para que tanto el usuario como el gimnasio puedan verlo después y hacer ajustes según sea necesario.

Por ahora, la app está en fase de prototipo, pero tenemos planeadas muchas más funciones para el futuro, como:

Calendario de rutinas para organizar los entrenamientos de cada usuario.

Temporizador de ejercicios, para que sea más fácil seguir las rutinas correctamente.

Calculadora de calorías para que cada persona pueda controlar su alimentación y objetivos de manera más precisa.

La idea es que esta aplicación sea una herramienta que haga que cada cliente del gimnasio se sienta acompañado, con un plan adaptado a sus necesidades y metas. Queremos combinar la tecnología con la atención personalizada, para que entrenar y cuidarse sea más fácil, seguro y motivador.
